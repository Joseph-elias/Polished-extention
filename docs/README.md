# Publish Privacy Policy (GitHub Pages)

1. Push this repository to GitHub.
2. In GitHub, open `Settings` -> `Pages`.
3. Under `Build and deployment`:
   - Source: `Deploy from a branch`
   - Branch: `main` (or your default branch)
   - Folder: `/docs`
4. Save and wait for GitHub Pages to publish.
5. Your policy URL will be:
   - `https://<your-github-username>.github.io/<repo-name>/privacy-policy.html`
6. Paste that URL into the Chrome Web Store `Privacy policy URL` field.

## Quick checklist
- Verify the page loads publicly in an incognito window.
- If you add a support email, update `privacy-policy.html` section 11.
